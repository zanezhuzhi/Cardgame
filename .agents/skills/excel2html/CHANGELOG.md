# 更新日志

## [3.1.0] - 2026-03-18

### ✨ 新增功能
- **智能文件选择系统**
  - 自动打开"待转换excel文件"文件夹供用户查看
  - 多文件友好选择界面，显示文件大小信息
  - 自动排除Excel临时文件（~$开头）
  - 单文件场景：快速确认模式
  - 多文件场景：带序号的列表选择
  
- **增强的用户交互**
  - 清晰的提示信息和进度反馈
  - 完整的输入验证和错误处理
  - 跨平台文件夹打开支持（Windows/macOS/Linux）
  - 支持默认值和快捷操作（直接回车）

- **安全防护升级**
  - XXE攻击防护：安全的XML解析器配置
  - XSS防护：所有用户内容HTML转义
  - 路径遍历防护：safe_path函数验证
  - 文件类型验证

### 🔧 改进
- 优化代码结构，修复html模块命名冲突
- 改进错误提示信息的友好度
