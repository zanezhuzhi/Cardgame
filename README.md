# 御魂传说 (Cardgame)

> 🎴 一款阴阳师主题的多人卡牌桌游，支持2-6人联机对战

## 项目简介

御魂传说是一款基于H5的策略卡牌游戏，玩家扮演阴阳师，召唤式神、收集御魂、退治妖怪，最终击败鬼王获得胜利。

## 开发指南

### 环境要求

- Node.js >= 18
- npm >= 9

### 安装依赖

```bash
# 共享模块
cd shared && npm install

# 服务端
cd server && npm install
```

### 运行测试

```bash
# 在 shared 目录下运行所有测试
cd shared && npm test

# 运行特定测试
npx vitest run ShikigamiSkills
npx vitest run YokaiEffects
npx vitest run BossEffects
```

### 启动服务端 (开发模式)

```bash
cd server && npm run dev
```

## 游戏规则概述

1. **准备阶段**：每位玩家选择阴阳师和2个初始式神
2. **回合流程**：
   - 鬼火阶段：获得1点鬼火
   - 式神调整阶段：可更换式神
   - 行动阶段：打出卡牌、使用技能、分配伤害
   - 清理阶段：弃置手牌、抽5张新牌
3. **胜利条件**：击败最终鬼王后，声望最高的玩家获胜

## 卡牌类型

| 类型 | 数量 | 说明 |
|------|------|------|
| 阴阳师 | 6张 | 玩家角色，各有专属技能 |
| 式神 | 24张 | 召唤物，提供启动/触发/被动技能 |
| 阴阳术 | 3种 | 基础伤害牌 |
| 徝魂(妖怪) | 38张 | 退治后可使用的御魂效果 |
| 鬼王 | 10张 | 阶段性Boss |
| 其他 | 若干 | 令牌、恶评等 |
