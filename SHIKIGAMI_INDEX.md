## 按类型分类

### 主动技能 (启动)
需要消耗鬼火或满足条件后主动使用

- #1 妖刀姬「杀戮」- 鬼火-2：抓牌+1，伤害+1（可追加）
- #3 酒吞童子「酒葫芦」- 鬼火-2：放置酒气指示物
- #5 花鸟卷「画境」- 鬼火-2：抓牌+3，置底1张
- #6 书翁「万象之书」- 鬼火-N：伤害+N+1
- #8 鬼使白「魂狩」- 鬼火-1：退治时可收入手牌
- #11 白狼「冥想」- 鬼火-1，弃置N张：伤害+N
- #12 食梦貘「沉睡」- 鬼火-1：抓牌+4，进入沉睡
- #14 萤草「生花」- 鬼火-1或弃妖怪：放置种子
- #16 食发鬼「真实之颜」- 鬼火-1：换场上妖怪
- #18 山童「怪力」- 鬼火-1：前2张术式伤害+1
- #21 青蛙瓷器「岭上开花」- 鬼火-1：掷骰，成功伤害+2
- #23 座敷童子「魂之火」- 弃妖怪牌：鬼火+1
- #24 山兔「兔子舞」- 鬼火-1：抓牌换牌

### 触发技能 (触发)

- #2 大天狗「羽刃暴风」- 溅射伤害

- #10 追月神「明月潮升」- 抓牌达3时选择奖励
- #17 巫蛊师「迷魂蛊」- 超度时交换手牌

### 被动技能 (永久/自动)

- #20 三尾狐「诱惑」- 对非女性首次伤害+1

### 妨害技能

- #22 铁鼠「横财护身」- 对手弃库顶，偷术式

## Review 检查点

每个式神技能在代码中应包含：
1. **序号和名称注释** - 如 `// #1 妖刀姬「杀戮」`
2. **技能描述** - 完整的规则文本
3. **鬼火消耗检查** - 使用 `payGhostFire()` 函数
4. **UI交互回调** - `onSelectCards`, `onChoice`, `onSelectTarget`
5. **错误处理** - 返回 `{ success: false, message: ... }`
6. **成功消息** - 返回 `{ success: true, message: ..., costPaid: ... }`
