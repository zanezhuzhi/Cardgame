## 🧪 手动测试指南

### 测试1：基础术式获取

**GM指令准备**：无需准备

**测试步骤**：
1. 进入游戏，轮到你的回合
2. 点击【获得阴阳术】按钮
3. 选择【基础术式】

**预期结果**：
- ✅ 弃牌堆+1（获得基础术式）
- ✅ 再次点击【获得阴阳术】→【基础术式】显示❌

---

### 测试2：中级符咒兑换

**GM指令准备**：
```bash
# 1. 获取房间和玩家ID
curl "http://localhost:3002/api/gm/debug"

# 2. 添加基础术式到手牌
curl "http://localhost:3002/api/gm/addcard/{roomId}/{playerId}/基础术式/1"

# 3. 添加妖怪到弃牌堆（需要hp≥2的妖怪）
curl "http://localhost:3002/api/gm/discard/{roomId}/{playerId}/灯笼鬼/1"
```

**测试步骤**：
1. 确认手牌有【基础术式】
2. 确认弃牌堆有妖怪（如灯笼鬼hp=2）
3. 点击【获得阴阳术】→【中级符咒】应显示✅
4. 点击后选择弃牌堆中的妖怪
5. 确认兑换

**预期结果**：
- ✅ 手牌中的基础术式消失
- ✅ 弃牌堆中的妖怪消失
- ✅ 超度区+2（基础术式和妖怪）
- ✅ 弃牌堆+1（获得中级符咒）
