## 一、架构概述

### 1.1 核心文件

| 文件 | 职责 |
|------|------|
| `shared/game/effects/BossEffects.ts` | 鬼王御魂效果与来袭效果 |
| `shared/game/effects/EffectEngine.ts` | 底层效果执行引擎 |
| `server/src/game/MultiplayerGame.ts` | 服务端鬼王系统处理 |

### 1.2 鬼王系统特点

与妖怪不同，鬼王具有以下特殊机制：

| 机制 | 说明 |
|------|------|
| **阶段系统** | 分为 Ⅰ/Ⅱ/Ⅲ 三个阶段，依次登场 |
| **来袭效果** | 登场时触发，影响所有玩家 |
| **御魂效果** | 作为御魂牌打出时触发 |
| **持续效果** | 部分鬼王具有【自】【触】被动效果 |

## 二、鬼王阶段系统

### 2.1 阶段划分

| 阶段 | 鬼王 | HP范围 | 声誉 |
|:----:|------|:------:|:----:|
| Ⅰ | 麒麟、石距、鬼灵歌伎、土蜘蛛 | 8-10 | +3~+4 |
| Ⅱ | 胧车、蜃气楼、荒骷髅 | 10-12 | +4 |
| Ⅲ | 地震鲶、八岐大蛇、贪嗔痴 | 13-15 | +5 |

### 2.2 登场流程

1. 击败当前鬼王后，进入清理阶段
2. 清理阶段结束时翻出下一张鬼王
3. 触发新鬼王的**来袭效果**
4. 来袭效果影响**所有玩家**（包括击败者）

## 三、效果类型

### 3.1 来袭效果

鬼王登场时触发，影响全体玩家：

| 鬼王 | 来袭效果 |
|------|----------|
| 麒麟 | 无来袭效果（首个鬼王） |
| 石距 | 所有玩家弃置1张手牌 |
| 鬼灵歌伎 | 所有玩家获得1张恶评 |
| 土蜘蛛 | 所有玩家弃置牌库顶1张 |
| 胧车 | 所有玩家弃置2张手牌 |
| 蜃气楼 | 所有玩家获得2张恶评 |
| 荒骷髅 | 所有玩家弃置牌库顶2张 |
| 地震鲶 | 所有玩家弃置3张手牌 |
| 八岐大蛇 | 所有玩家获得3张恶评 |
| 贪嗔痴 | 所有玩家弃置牌库顶3张 |

### 3.2 御魂效果

鬼王作为御魂牌打出时触发：

| 鬼王 | 御魂效果 |
|------|----------|
| 麒麟 | 伤害+3，【触】回合结束时从弃牌堆回到牌库底 |
| 石距 | 伤害+3，选择1名对手弃置1张手牌 |
| 鬼灵歌伎 | 伤害+3，选择1名对手获得恶评 |
| 土蜘蛛 | 伤害+4，选择1名对手弃置牌库顶 |
| 胧车 | 伤害+4，选择1名对手弃置2张手牌 |
| 蜃气楼 | 伤害+4，选择1名对手获得2张恶评 |
| 荒骷髅 | 伤害+4，选择1名对手弃置牌库顶2张 |
| 地震鲶 | 伤害+5，所有对手弃置1张手牌 |
| 八岐大蛇 | 伤害+5，所有对手获得1张恶评 |
| 贪嗔痴 | 伤害+5，所有对手弃置牌库顶1张 |

### 3.3 特殊效果

| 鬼王 | 效果类型 | 效果说明 |
|------|----------|----------|
| 麒麟 | 【触】 | 回合结束时若在弃牌堆，回到牌库底 |

## 四、交互系统

### 4.1 来袭效果交互

来袭效果涉及的玩家选择：

| 效果 | 交互类型 | 说明 |
|------|----------|------|
| 弃置手牌 | selectHandCards | 每位玩家依次选择 |
| 获得恶评 | 无 | 自动从恶评牌库抽取 |
| 弃置牌库顶 | 无 | 自动执行 |

### 4.2 御魂效果交互

| 效果 | 交互类型 | 说明 |
|------|----------|------|
| 选择对手 | selectPlayer | 选择1名其他玩家 |
| 对手弃牌 | selectHandCards | 被选中对手选择 |

### 4.3 妨害抵抗

鬼王的妨害效果可被式神抵抗（花鸟卷、萤草等）。

### 6.2 来袭效果执行顺序

来袭效果按以下顺序执行：

1. 从**当前玩家**开始
2. 按**顺时针顺序**依次处理
3. 每位玩家的选择**依次进行**（非同时）

### 6.3 与妖怪效果的区别

| 差异点 | 妖怪 | 鬼王 |
|--------|------|------|
| 登场效果 | 无 | 来袭效果 |
| 御魂基础伤害 | 0~2 | 3~5 |
| 御魂目标 | 多样 | 多为妨害 |
| 持续效果 | 少见 | 麒麟【触】 |
