## 二、鬼王阶段系统

### 2.1 阶段划分

| 阶段 | 鬼王 | HP | 声誉 |
|:----:|------|:--:|:----:|
| Ⅰ | 麒麟、石距、鬼灵歌伎🔷、土蜘蛛 | 8-10 | +3~+4 |
| Ⅱ | 胧车、蜃气楼、荒骷髅🔷 | 10-12 | +4 |
| Ⅲ | 地震鲶、八岐大蛇、贪嗔痴🔷 | 13-15 | +5 |

### 2.2 登场流程（重要！）

```
清理阶段流程：
1. 弃置已打出牌 + 剩余手牌
2. 【御魂/持续】类型卡可选择抓回手上
3. 从牌库抽取 5 张（下回合手牌）
4. 补充妖怪至 6 张
5. ★ 如果本回合击杀了鬼王 → 翻出新鬼王 → 执行【鬼王来袭】
6. 回合结束，清空伤害
```

**关键点**：
- 鬼王翻出在**抽牌后**，所以来袭效果影响的是**已抽好的下回合手牌**
- 来袭效果影响**所有玩家**（包括击败者）
- 从**当前玩家**开始，按**顺时针顺序**依次处理

### 2.3 多人专属标记

## 三、效果类型（以鬼王卡.md为准）

### 3.1 来袭效果

> 鬼王登场时触发，影响**所有玩家**

| 鬼王 | 阶段 | 来袭效果 |
|------|:----:|----------|
| 麒麟 | Ⅰ | **无**（首个鬼王） |
| 石距 | Ⅰ | 每位玩家展示手牌，弃掉所有阴阳术。未弃牌者获得1张恶评 |
| 鬼灵歌伎🔷 | Ⅰ | 每位玩家展示牌库顶5张牌，将所有HP>6的卡牌弃置，其他以任意顺序放回牌库顶 |
| 土蜘蛛 | Ⅰ | 每位玩家从手牌展示3张阴阳术，每缺1张则弃1张手牌 |
| 胧车 | Ⅱ | 每位玩家展示手牌并超度1张御魂牌。无法执行则获得1张恶评 |
| 蜃气楼 | Ⅱ | 每位玩家将手牌中HP>6的卡牌弃置 |
| 荒骷髅🔷 | Ⅱ | 每位玩家将整个牌库弃置，从弃牌区超度1张HP>7的御魂卡，并获得1张恶评。然后重洗牌库 |
| 地震鲶 | Ⅲ | 每位玩家在清理阶段补充完手牌后，随机将1张手牌背面朝上放置于阴阳师下。此鬼王被击败后将阴阳师下的所有卡牌弃置 |
| 八岐大蛇 | Ⅲ | 每位玩家按顺序展示手牌，弃掉HP最高的1张手牌。所有式神翻面失去能力直至大蛇离场 |
| 贪嗔痴🔷 | Ⅲ | 每位玩家随机弃掉1张手牌，弃掉生命值最高的1位或多位玩家再弃置1张手牌 |

### 3.2 御魂效果

> 鬼王作为御魂牌打出时触发

| 鬼王 | 阶段 | 御魂效果 |
|------|:----:|----------|
| 麒麟 | Ⅰ | 伤害+3。**【触】** 回合结束时，若此牌在弃牌区，将其置入牌库底 |
| 石距 | Ⅰ | 鬼火+1，抓牌+1，伤害+2 |
| 鬼灵歌伎🔷 | Ⅰ | 所有对手选择：随机弃掉1张手牌，或你抓牌+1 |
| 土蜘蛛 | Ⅰ | 抓牌+2，伤害+3 |
| 胧车 | Ⅱ | 抓牌+1，伤害+3，将任意数量的「游荡妖怪」置入牌库底，并立刻补充相同数量的牌 |
| 蜃气楼 | Ⅱ | 抓牌+1，鬼火+1，伤害+1。**【自】** 回合开始时，若此牌在弃牌堆中，将其置入手牌 |
| 荒骷髅🔷 | Ⅱ | 抓牌+1。本回合中你可以将所有卡视作「中级符咒」来使用。**【自】** 回合开始时，若此牌在弃牌堆中，将其置入手牌 |
| 地震鲶 | Ⅲ | 所有对手玩家将手牌数弃至3张，你可以获得其中HP最高的御魂卡并置入手牌（同为最高则你可选择） |
| 八岐大蛇 | Ⅲ | 鬼火+2，伤害+7 |
| 贪嗔痴🔷 | Ⅲ | 抓牌+1，超度1张手牌，选择1个妖怪或鬼王，对其造成等同超度牌HP的伤害。**【自】** 回合开始时，若此牌在弃牌堆中，将其置入手牌 |

### 3.3 特殊被动效果

| 鬼王 | 效果类型 | 效果说明 |
|------|----------|----------|
| 麒麟 | **【触】** | 回合结束时若在弃牌堆，回到牌库底 |
| 蜃气楼 | **【自】** | 回合开始时若在弃牌堆，回到手牌 |
| 荒骷髅 | **【自】** | 回合开始时若在弃牌堆，回到手牌 |
| 贪嗔痴 | **【自】** | 回合开始时若在弃牌堆，回到手牌 |

### 6.2 来袭效果执行顺序

来袭效果按以下顺序执行：

1. 从**当前玩家**（击败鬼王者）开始
2. 按**顺时针顺序**依次处理
3. 每位玩家的选择**依次进行**（非同时）

### 6.3 与妖怪御魂的区别

| 差异点 | 妖怪御魂 | 鬼王御魂 |
|--------|----------|----------|
| 登场效果 | 无 | 来袭效果 |
| 徝魂基础伤害 | 0~2 | 0~7 |
| 徝魂效果 | 多样 | 多为高数值或妨害 |
| 持续效果 | 少见 | 麒麟【触】、蜃气楼/荒骷髅/贪嗔痴【自】 |

### 6.4 鬼王翻出时机（已修复）

```typescript
// MultiplayerGame.ts - defeatBoss()
// 击败鬼王时设置标记，不立即翻出
(this.state as any).pendingBossReveal = true;

// MultiplayerGame.ts - enterCleanupPhase()
// 在抽牌后、补妖怪后检查标记
if ((this.state as any).pendingBossReveal) {
  (this.state as any).pendingBossReveal = false;
  this.revealBoss(); // 此时翻出新鬼王
}
```
