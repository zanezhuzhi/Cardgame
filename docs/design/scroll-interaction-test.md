## 手动验收步骤

### 步骤1: 基础滚动功能
1. 启动游戏，进入房间
2. 执行多次操作生成足够多的日志（>20条）
3. 验证滚动条出现且可拖拽
4. 验证鼠标滚轮可滚动

### 步骤2: 消息限制测试
1. 使用GM指令批量生成日志：`/testlog 350`
2. 验证日志区最多显示300条
3. 验证最早的50条被删除

### 步骤3: 智能滚动测试
1. 手动滚动到历史消息区域
2. 执行操作生成新日志
3. 验证页面不自动滚动，显示"↓ 新消息"按钮
4. 点击按钮，验证滚动到底部
