## 式神获取/置换规则

### 获得式神（式神<3时）
**条件**：
- 持有式神 < 3 个
- 手牌中有高级符咒（3点伤害）
- 符咒总伤害 ≥ 5 点

**流程**：
1. 选择符咒超度（≥5点，必须含高级符咒）
2. 从式神商店随机抽2张，选择1张获得

### 置换式神（式神=3时）
**条件**：
- 持有式神 = 3 个
- 手牌中有高级符咒（3点伤害）

**流程**：
1. 选择1张高级符咒超度
2. 从式神商店随机抽2张，选择1张作为新式神
3. 选择1个现有式神替换掉（返回商店底部）
4. 新式神加入
