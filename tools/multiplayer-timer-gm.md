# 多人计时 / 托管 — GM 手测清单

在对局内**聊天输入框**输入以下指令（以 `/` 开头）。实现位置：`server/src/socket/SocketServer.ts` 的 `executeGMCommand`。

## 指令一览

| 指令 | 作用 |
|------|------|
| `/help` | 列出当前可用 GM 命令摘要 |
| `/timeout` | 立即触发**当前行动玩家**的回合超时结算（先收口 `pendingChoice`，再结束回合） |
| `/afk [秒]` | 将**自己**的最后操作时间回拨指定秒数（默认 301）并立刻执行 AFK 判定 |
| `/offline` | 模拟本机掉线，进入离线托管（180 秒保护窗口，由游戏逻辑驱动） |
| `/online` | 模拟本机恢复在线，取消离线托管，恢复真人控制 |
| `/status` | 当前房间状态摘要 |
| `/ping` | 服务器时间 |
| `/addcard <卡牌名> [数量]` | 向自己手牌添加测试牌 |

## 最短验收路径（约 2～3 分钟）

1. 开局进入 **playing + action** 后输入 `/timeout`，确认回合自动结束、不卡弹窗。
2. 输入 `/offline`，日志应出现掉线/托管相关提示，该座位可由托管逻辑行动。
3. 输入 `/online`，确认恢复真人控制。
4. 输入 `/afk 301`，确认触发长时间无操作退出。
