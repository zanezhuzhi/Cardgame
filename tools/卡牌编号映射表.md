## 使用说明

### 步骤 1: 准备图片
将你的原始图片按顺序命名（01.png, 02.png...），放入对应目录：
```
tools/input/
├── boss/        # 01.png ~ 10.png (鬼王)
├── yokai/       # 01.png ~ 39.png (妖怪)
├── shikigami/   # 01.png ~ 24.png (式神)
├── spell/       # 01.png ~ 03.png (阴阳术)
└── curse/       # 01.png ~ 02.png (恶评)
```

### 步骤 2: 安装依赖
```bash
pip install Pillow
```

### 步骤 3: 运行处理脚本
```bash
cd D:\Cardgame\tools
python process_images.py
```

### 步骤 4: 检查输出
处理后的图片会输出到统一目录，按编号命名：
```
client/src/assets/images/cards/
├── boss/        # 101.png ~ 110.png (@2x: 390×280)
├── yokai/       # 201.png ~ 239.png (@2x: 280×390)
├── shikigami/   # 401.png ~ 424.png (@2x: 280×390)
├── spell/       # 601.png ~ 603.png (@2x: 280×390)
└── curse/       # 701.png ~ 702.png (@2x: 280×390)
```

## 输出尺寸配置

### 万智牌标准比例
- **实体尺寸**: 63mm × 88mm
- **长宽比**: 63:88 ≈ **5:7** (0.716)
- **适配线下印刷**: ✅

### @2x 高分辨率输出
- **倍率**: 2x（缩放不模糊）
- **UI基准**: 140×195 / 195×140
- **实际输出**: 280×390 / 390×280

### 排版规则
| 类型 | 排版方向 | 说明 |
|------|:--------:|------|
| 🎴 鬼王卡 | **横向** | 宽 > 高，类似横版名片 |
| 🃏 其他卡 | **竖向** | 高 > 宽，标准扑克牌方向 |

### 尺寸对照表

| 类型 | 编号段 | 排版 | 输出尺寸 (@2x) | 输出目录 |
|------|:------:|:----:|:--------------:|----------|
| 鬼王 | 1XX | 横向 | **390×280** | `cards/boss/` |
| 妖怪 | 2XX | 竖向 | **280×390** | `cards/yokai/` |
| 式神 | 4XX | 竖向 | **280×390** | `cards/shikigami/` |
| 阴阳术 | 6XX | 竖向 | **280×390** | `cards/spell/` |
| 恶评 | 7XX | 竖向 | **280×390** | `cards/curse/` |

> 💡 **设计说明**:
> - 所有卡牌统一基准尺寸 **140px**，输出 **@2x** 高清
> - 只有鬼王卡是**横向**排版
> - 式神卡在UI中显示较小，但**悬浮时显示原尺寸**
> - 编号段预留 3XX、5XX 用于未来扩展

如需调整尺寸，修改 `tools/process_images.py` 中的：
```python
CARD_WIDTH_BASE = 140   # UI显示基准宽度
SCALE_FACTOR = 2        # 输出倍率（@2x）
```
