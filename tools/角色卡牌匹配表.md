# 🎴 角色卡牌手动匹配表

> 请根据图片内容，填写对应的输出编号
> 完成后运行 `python match_cards.py` 自动复制并重命名

## 使用方法
1. 打开 `D:\Cardgame\策划文档\角色卡牌\` 查看每张图片
2. 对照 `D:\Cardgame\策划文档\打印卡牌\` 找到对应卡牌名称
3. 在下表中填写输出编号
4. 运行脚本自动处理
