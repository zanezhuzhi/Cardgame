## 零、视觉风格定义

### 整体风格
- **主题**：日式和风、阴阳师、樱花、神社
- **色调**：深紫蓝色背景 + 粉紫色樱花点缀 + 金色边框装饰
- **氛围**：神秘、优雅、东方奇幻

### 背景设计
- **主色调**：深紫蓝渐变 (`#1a1a2e` → `#2d1f3d` → `#1e3a5f`)
- **装饰元素**：樱花飘落、神社鸟居、远山剪影、烟雾缭绕
- **光效**：柔和的蓝紫色光晕、金色火焰

### 卡牌设计
- **边框**：金色/铜色金属质感边框，带装饰性花纹
- **卡背**：深色底 + 阴阳图案/御魂纹样
- **卡面**：精美立绘 + 底部信息栏（名称、数值）

### UI元素风格
- **面板**：半透明磨砂玻璃效果 + 金色边框
- **按钮**：圆角矩形 + 金色描边 + 悬停发光
- **文字**：标题用书法字体，正文用简洁衬线体
- **图标**：日式风格图标（鬼火、符咒、�的魂等）

## 一、伤害分配系统

### 1.1 核心规则
根据游戏规则说明书 6.1 节：
- **实时分配**：行动阶段内可随时分配伤害，不限次数
- **伤势保留**：已分配的伤害在回合内持续生效（目标生命扣减保留）
- **可随时切换目标**：不限制"必须击杀当前目标才能选新目标"

### 1.2 妖怪卡片视觉状态

| 状态 | 边框颜色 | 背景 | 说明 |
|------|----------|------|------|
| **正常** | 默认蓝色 | rgba(100,150,200,.3) | 未受伤的妖怪 |
| **受伤** | 淡红色 `#ff6b6b` | rgba(255,107,107,.15) | 当前HP < 最大HP |
| **可击杀** | 绿色 `#4CAF50` + 发光 | — | 玩家伤害 ≥ 妖怪剩余HP |
| **已击杀** | 粉红色 `#e91e63` + 发光 | rgba(233,30,99,.2) | HP ≤ 0，等待退治/超度 |

### 1.3 HP显示
- 格式：`❤️当前HP/最大HP`
- 受伤时数字变为红色 `#ff6b6b`
- 已击杀时显示 `💀 已击杀` 徽章

### 1.4 交互流程

```
点击妖怪卡片
    │
    ├── 妖怪已击杀？
    │       ├── 是 → 弹出选择框：「退治」或「超度」
    │       │           ├── 退治 → 放入弃牌堆（计声誉）
    │       │           └── 超度 → 移出游戏（不计声誉）
    │       │
    │       └── 否 → 玩家有伤害？
    │               ├── 有 → 分配伤害（最多到击杀）
    │               │         └── HP归零？显示「已击杀」
    │               └── 无 → 显示提示「没有可分配的伤害」
    │
回合结束
    └── 自动处理：
        ├── 已击杀未处理 → 自动退治
        └── 存活妖怪 → HP恢复满
```

## 二、式神技能系统

### 2.1 被动技能触发

#### 鲤鱼精「泡泡之盾」
- **触发时机**：回合内首次退治妖怪或鬼王时
- **效果**：可选择将目标放到牌库顶而非弃牌堆
- **交互**：弹出选择框，玩家确认是否使用
- **限制**：每回合仅触发一次

## 四、卡牌使用规则

### 4.1 可打出的卡牌类型

| 卡牌类型 | 可否打出 | 说明 |
|----------|----------|------|
| 阴阳术 (spell) | ✅ 可以 | 造成伤害 |
| 御魂 (yokai) | ✅ 可以 | 触发御魂效果 |
| 鬼王御魂 (boss) | ✅ 可以 | 触发鬼王御魂效果 |
| 令牌 (token) | ❌ 不可 | 招福达摩无法打出，声誉+1，仅用于超度 |
| 恶评 (penalty) | ❌ 不可 | 恶评卡无法打出 |
| 式神 (shikigami) | ❌ 不可 | 需要通过召唤，不能直接打出 |

### 4.2 御魂效果目标要求

**核心规则**：
- **单效果卡**：唯一效果需要合法目标才能打出，否则无法使用
- **多效果卡**：只要任一效果能执行即可打出，无法选择目标的效果自动跳过

#### 单效果卡（必须有目标）

| 御魂名称 | 目标要求 | 无目标时 |
|----------|----------|----------|
| 天邪鬼绿 | 场上有生命≤4的游荡妖怪 | ❌ 不可打出 |
| 海坊主 | 场上有生命≤6的游荡妖怪 | ❌ 不可打出 |
| 提灯小僧 | 场上有游荡妖怪 | ❌ 不可打出 |
| 古笼火 | 场上有游荡妖怪 | ❌ 不可打出 |
| 一目连 | 弃牌堆不为空 | ❌ 不可打出 |
| 独眼小僧 | 牌库不为空 | ❌ 不可打出 |

#### 多效果卡（有保底效果）

| 御魂名称 | 效果1（保底） | 效果2（可选） |
|----------|---------------|---------------|
| 唐纸伞妖 | 伤害+1 ✅ | 查看牌库顶（需牌库）|
| 天邪鬼赤 | 伤害+1 ✅ | 换牌（需手牌）|
| 天邪鬼黄 | 抓牌+2 ✅ | 置顶1张（有手牌时）|
| 树妖 | 抓牌+2 ✅ | 弃置1张（需手牌）|
| 河童 | 伤害+2 ✅ | 弃置换牌（需手牌）|
| 雪女 | 伤害+1 ✅ | 弃置抓牌（需手牌）|
| 座敷童子 | 抓牌+1 ✅ | 从弃牌堆取回（需弃牌）|

### 4.3 不可打出卡牌的视觉表现
- 半透明度 (opacity: 0.5)
- 轻微灰度滤镜
- 鼠标悬停无抬起效果
- 光标变为禁止样式

### 4.3 各类型卡牌颜色

| 类型 | 背景渐变色 |
|------|-----------|
| 阴阳术 | 蓝色 `#2196F3` → `#1976D2` |
| 御魂 | 绿色 `#4CAF50` → `#388E3C` |
| 令牌 | 橙色 `#FFB74D` → `#FF9800` |
| 恶评 | 灰色 `#78909C` → `#546E7A` |
| 鬼王 | 紫色 `#9C27B0` → `#7B1FA2` |
