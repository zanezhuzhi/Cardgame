## 2026-03-21

### 关于美术团队工作流设计

希望创建一个由7个职能 Agent 组成的美术团队：
- 交互UI设计师
- GUI设计师
- 资产管理师
- 美术设计师
- 动效设计师
- 美术总监
- 制作人助手

工作流程设计：
- 制作人提供设计方案 → 交互UI设计师完善界面交互细节，提交需求文档
- 制作人认可后 → GUI设计师评估交互设计细节和设计效果
- GUI设计效果 → 美术总监 + 制作人助手交叉审核
- 交叉审核后 → 制作人确认进入下一环节
- 资产管理师拆解美术资源需求列表、资源规格及使用细节
- 将需求发给美术设计师和动效设计师
- 需求拆解完成 → 再次经美术总监 + 制作人助手交叉审核
- 制作人认可后 → 美术设计师 + 动效设计师产出资源
- 产出资源 → 资产管理师规格确认后输出给制作人

每个 Agent 的核心描述由制作人和 AI 共同创建。

### 关于界面布局设计（Figma）

采用 1920×1080 横屏分辨率作为主设计尺寸。

卡牌比例参考万智牌（MTG）标准：5:7（63mm×88mm）。

竖版卡牌尺寸：140×195（手牌/式神），商店展示可等比放大。

顶部进度区 150px，玩家头像约 120×120。

三区布局：顶部（进度）、中间（共享对战区）、底部（个人区）。

右侧区域放操作日志（从上到下刷新的信息列表）+ 三个交互按钮（获得式神、获得阴阳术、查看超度区）。
