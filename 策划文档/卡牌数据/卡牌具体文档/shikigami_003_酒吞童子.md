---

## 🔍 效果分解

### A. 「储酒」：超度换酒气

1. **支付鬼火**：消耗 2 点鬼火
2. **超度 1 张手牌**：将 1 张手牌移出游戏（进入公共超度区）
3. **放置 1 枚酒气**：在该式神上增加 1 枚「酒气」指示物（上限 3）

### B. 「释放」：酒气换伤害

1. **选择移除数量 N**：选择移除 \(N\) 枚酒气（\(1 \le N \le\) 当前酒气数）
2. **移除酒气**：移除 \(N\) 枚酒气
3. **伤害 +N**：本回合可分配伤害增加 \(N\)

### 关键规则

- 酒气上限为 3：当酒气已达 3 枚时，**「储酒」无法启动**（不能再支付鬼火-2去超度换酒气）。
- 本技能点击发动时，先进行 **2 选 1**：选择「储酒」或「释放」，再进入对应的后续选择（例如：储酒选择要超度的手牌；释放选择移除的酒气数量 \(N\)）。

| 决策点 | AI策略（L1规则） |
|--------|------------------|
| 何时储酒 | 当手牌中存在低价值牌可超度且预计后续需要爆发伤害时优先储酒 |
| 超度哪张手牌 | 优先超度对当前局面价值最低的牌（通常是低伤害阴阳术或低收益御魂） |
| 何时释放 | 当本回合存在明确击杀线（可击杀关键游荡妖怪/鬼王）时释放 |
| N 取值 | 优先选择“刚好足够击杀”的最小 N，避免浪费酒气 |

## ✅ 验收标准

- [ ] 储酒：鬼火-2 后必须超度 1 张手牌，且酒气最多到 3
- [ ] 释放：移除 N 枚酒气后，伤害正确增加 N
- [ ] 酒气不会超过上限 3
- [ ] 强制中止/超时：储酒默认不超度（若必须选则选第一张），释放默认选择 N=1
