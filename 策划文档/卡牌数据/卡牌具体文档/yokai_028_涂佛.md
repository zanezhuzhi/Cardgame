## 🔍 效果分解

### 步骤拆解

1. **检查弃牌区**：查看玩家弃牌区中的阴阳术卡牌
2. **选择阴阳术**：玩家选择最多3张阴阳术
3. **置入手牌**：将选中的阴阳术从弃牌区移到手牌

### 关键规则

- **选择数量**：最多3张，弃牌区不足3张时选择全部
- **卡牌类型**：仅限阴阳术（基础术式、中级符咒、高级符咒）
- **无声誉**：退治此卡不获得声誉（charm = 0）
- **资源回收**：属于"资源回收"类效果，恢复已使用的阴阳术

### 阴阳术卡牌速查

| 卡牌名 | 效果 |
|--------|------|
| 基础术式 | 伤害+1 |
| 中级符咒 | 伤害+2 |
| 高级符咒 | 伤害+3 |

## 🛠️ 实现备注

### 涉及文件

- `shared/game/effects/YokaiEffects.ts` - 效果实现

### 关键函数

```typescript
registerEffect('涂佛', async (ctx) => {
  const { player, triggerInteract } = ctx;
  
  // 1. 筛选弃牌区中的阴阳术
  const spellCards = player.discard.filter(card => card.cardType === 'spell');
  
  if (spellCards.length === 0) {
    return { success: true, message: '弃牌区无阴阳术' };
  }
  
  // 2. 玩家选择最多3张
  const maxSelect = Math.min(3, spellCards.length);
  const choice = await triggerInteract?.({
    type: 'selectCards',
    playerId: player.id,
    cards: spellCards,
    min: 0,
    max: maxSelect,
    prompt: `选择最多${maxSelect}张阴阳术置入手牌`
  });
  
  // 3. 移入手牌
  const selectedIds = choice?.selectedCardIds || [];
  for (const id of selectedIds) {
    const index = player.discard.findIndex(c => c.instanceId === id);
    if (index !== -1) {
      const card = player.discard.splice(index, 1)[0];
      player.hand.push(card);
    }
  }
  
  return { success: true, message: `回收${selectedIds.length}张阴阳术` };
});
```

### 注意事项

- 需要正确区分 `cardType === 'spell'`
- 选择界面需要显示阴阳术的具体名称和效果
- 单人模式AI需要自动选择（优先高伤害）
