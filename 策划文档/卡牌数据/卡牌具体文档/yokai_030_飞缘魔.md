## 🔍 效果分解

### 步骤拆解

1. **打出飞缘魔** → 触发御魂效果
2. **获取当前鬼王** → 检查场上存在的鬼王
3. **执行鬼王御魂效果** → 复制并执行该鬼王的御魂效果

### 关键规则

- **引用当前鬼王**：使用场上**当前存在**的鬼王的御魂效果
- **完整复制**：完整执行鬼王御魂效果，不打折扣
- **无自身加成**：飞缘魔本身无伤害/鬼火加成，效果完全来自鬼王
- **鬼王不存在时**：如果所有鬼王已被击败（游戏即将结束），无效果可用

## ⚔️ 交互场景

### 场景1：使用麒麟御魂（阶段Ⅰ）

```
当前状态:
- 场上鬼王: 麒麟 (HP8)
- 玩家 打出 飞缘魔

执行流程:
1. 获取当前鬼王 → 麒麟
2. 执行麒麟御魂效果 → 伤害+3
3. 玩家伤害: 0 → 3

结果:
- 伤害+3
- 注：麒麟的【触】回合结束回库效果不触发（那是麒麟本体的效果）
```

### 场景2：使用八岐大蛇御魂（阶段Ⅲ）

```
当前状态:
- 场上鬼王: 八岐大蛇 (HP14)
- 玩家 打出 飞缘魔

执行流程:
1. 获取当前鬼王 → 八岐大蛇
2. 执行八岐大蛇御魂效果 → 鬼火+2，伤害+7
3. 玩家鬼火: 2 → 4
4. 玩家伤害: 0 → 7

结果:
- 鬼火+2
- 伤害+7（最强御魂效果）
```

### 场景3：使用石距御魂（综合资源型）

```
当前状态:
- 场上鬼王: 石距 (HP9)
- 玩家 打出 飞缘魔

执行流程:
1. 获取当前鬼王 → 石距
2. 执行石距御魂效果 → 鬼火+1，抓牌+1，伤害+2
3. 玩家鬼火: 2 → 3
4. 玩家从牌库抓1张牌
5. 玩家伤害: 0 → 2

结果:
- 鬼火+1
- 抓牌+1
- 伤害+2
```

### 场景4：鬼王已全部击败（边界）

```
当前状态:
- 场上鬼王: 无（所有鬼王已被击败，游戏即将结束）
- 玩家 打出 飞缘魔

执行流程:
1. 获取当前鬼王 → 无
2. 无御魂效果可执行

结果:
- 无效果
- 提示: "当前无鬼王存在"
```

## 🛠️ 实现备注

### 涉及文件

- `shared/game/effects/YokaiEffects.ts` - 飞缘魔效果实现
- `shared/game/effects/BossEffects.ts` - 鬼王御魂效果定义
- `shared/types/game.ts` - 鬼王状态类型

### 关键函数

```typescript
registerEffect('飞缘魔', async (ctx) => {
  const { player, gameState } = ctx;
  
  // 1. 获取当前鬼王
  const currentBoss = gameState.currentBoss;
  
  if (!currentBoss) {
    ctx.addLog?.(`⚠️ 当前无鬼王存在`);
    return { success: true, message: '无鬼王可用' };
  }
  
  // 2. 获取鬼王御魂效果
  const bossName = currentBoss.name;
  ctx.addLog?.(`🎴 飞缘魔 → 使用 ${bossName} 的御魂效果`);
  
  // 3. 执行鬼王御魂效果（复用 BossEffects.ts 中的实现）
  const result = await executeBossSoulEffect(bossName, ctx);
  
  return result;
});

// 注：executeBossSoulEffect 定义在 BossEffects.ts 中
// 鬼王御魂效果数据来源：策划文档/卡牌数据/鬼王卡.md
```

### 注意事项

- **仅复制御魂效果**：不复制鬼王的【触】【自】【永】等被动效果
- **实时获取**：使用时获取当前鬼王，而非打出时
- **鬼王效果独立**：需要维护鬼王御魂效果的映射表
- **多人标记**：此卡有1张多人专属（5-6人游戏使用2张）
