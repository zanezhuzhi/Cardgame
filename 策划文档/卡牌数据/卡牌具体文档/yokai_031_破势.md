## 🔍 效果分解

### 步骤拆解

1. **判断首张** → 检查本回合是否已使用过其他牌
2. **计算伤害** → 首张则+5，否则+3
3. **增加伤害** → 将计算结果加到玩家伤害池

### 关键规则

- **首张判定**：检查 `cardsPlayed` 数组，打出此牌时若为空则视为首张
- **首张加成**：首张时伤害+5，否则伤害+3
- **差值收益**：首张比非首张多+2伤害
- **无条件触发**：无论是否首张都会产生伤害

### 首张判定逻辑

| 条件 | 伤害 | 说明 |
|------|:----:|------|
| `cardsPlayed.length === 0` | +5 | 本回合第一张牌 |
| `cardsPlayed.length > 0` | +3 | 已使用过其他牌 |

## 🎲 AI 行为决策

### AI 使用破势的时机

| 决策点 | AI策略 | 优先级说明 |
|--------|--------|-----------|
| 首张时 | **最高优先** | 伤害+5，收益最大化 |
| 非首张时 | 正常优先 | 伤害+3，仍有价值 |

### AI 开局策略

```

  优先作为首张打出
否则:
  按正常策略打牌
```

## 🛠️ 实现备注

### 涉及文件

- `shared/game/effects/YokaiEffects.ts` - 破势效果实现
- `shared/types/game.ts` - PlayerState.cardsPlayed

### 关键函数

```typescript
registerEffect('破势', async (ctx) => {
  const { player } = ctx;
  
  // 判断是否为首张
  const isFirstCard = player.cardsPlayed.length === 0;
  
  // 计算伤害
  const damage = isFirstCard ? 5 : 3;
  
  // 增加伤害
  player.damage += damage;
  
  // 日志
  if (isFirstCard) {
    ctx.addLog?.(`🎴 破势（首张）→ 伤害+${damage}`);
  } else {
    ctx.addLog?.(`🎴 破势 → 伤害+${damage}`);
  }
  
  return { success: true, message: `伤害+${damage}` };
});
```

### 注意事项

- **cardsPlayed 维护**：在卡牌打出后、效果执行前，需先将卡牌加入 `cardsPlayed`
- **首张判定时机**：在效果执行时判断，此时当前卡尚未加入 `cardsPlayed`
- **与轮入道配合**：效果执行两次，但首张判定只检查一次
