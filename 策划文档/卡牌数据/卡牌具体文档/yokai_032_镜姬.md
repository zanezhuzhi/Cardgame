## 📜 效果原文

> 抓牌+2，伤害+1，鬼火+1。【妖】【永】此牌不会受到由阴阳术产生的伤害。

---

## 🔍 效果分解

### 步骤拆解

1. **抓牌** → 从牌库抓2张牌
2. **增加伤害** → 伤害+1
3. **增加鬼火** → 鬼火+1（上限5）

### 关键规则

- **综合资源型**：同时提供抓牌、伤害、鬼火三种资源
- **【妖】标记**：仅在作为游荡妖怪时生效
- **【永】持续效果**：只要在游荡区就持续生效
- **阴阳术免疫**：作为游荡妖怪时，不受阴阳术（基础术式/中级符咒/高级符咒）产生的伤害

### 【妖】【永】效果详解

| 状态 | 【妖】效果 |
|------|----------|
| 在游荡区 | **生效** - 免疫阴阳术伤害 |
| 在手牌中 | 不生效 |
| 打出后 | 不生效 |

### 免疫范围

| 伤害来源 | 是否免疫 |
|----------|:--------:|
| 基础术式 | ✅ 免疫 |
| 中级符咒 | ✅ 免疫 |
| 高级符咒 | ✅ 免疫 |
| 御魂效果产生的伤害 | ❌ 不免疫 |
| 式神技能产生的伤害 | ❌ 不免疫 |

## 🛠️ 实现备注

### 涉及文件

- `shared/game/effects/YokaiEffects.ts` - 御魂效果实现
- `server/src/game/MultiplayerGame.ts` - 【妖】效果判定
- `shared/types/game.ts` - 伤害来源类型

### 关键函数

```typescript
// 御魂效果
registerEffect('镜姬', async (ctx) => {
  const { player } = ctx;
  
  // 抓牌+2
  await drawCards(player, 2, ctx);
  
  // 伤害+1
  player.damage += 1;
  
  // 鬼火+1
  player.ghostFire = Math.min(player.ghostFire + 1, 5);
  
  ctx.addLog?.(`🎴 镜姬 → 抓牌+2，伤害+1，鬼火+1`);
  
  return { success: true, message: '抓牌+2，伤害+1，鬼火+1' };
});

// 【妖】效果判定（在伤害分配时）
function canDamageTarget(target: CardInstance, damageSource: DamageSource): boolean {
  // 镜姬【妖】效果：免疫阴阳术伤害
  if (target.name === '镜姬' && damageSource.type === 'spell') {
    return false;
  }
  return true;
}
```

### 注意事项

- **多人专属**：此卡仅在5-6人游戏使用（`multiPlayer = count = 2`）
- **伤害来源追踪**：需要区分伤害来源类型（阴阳术/御魂/式神技能）
- **【妖】效果位置**：仅在游荡区生效，手牌中不生效
