## 📜 效果原文

> 从你的牌库顶开始展示卡牌，直到出现3张阴阳术，将它们置入手牌，弃置其余展示的卡牌。

---

## 🔍 效果分解

### 步骤拆解

1. **开始展示** → 从牌库顶逐张翻开展示
2. **检查阴阳术** → 统计已展示的阴阳术数量
3. **达到3张时停止** → 或牌库翻完
4. **阴阳术入手** → 将展示的阴阳术置入手牌
5. **其余弃置** → 其他展示的牌进入弃牌堆

### 关键规则

- **搜索型效果**：从牌库中筛选特定类型的牌
- **阴阳术范围**：基础术式、中级符咒、高级符咒、专属符咒
- **展示公开**：所有展示的牌对全体玩家可见
- **最多3张**：找到3张阴阳术即停止
- **可能不足3张**：牌库中阴阳术不足3张时，有几张拿几张
- **弃置触发**：被弃置的牌可能触发【被弃置时】效果

### 阴阳术类型

| 类型 | 是否计入 |
|------|:--------:|
| 基础术式 | ✅ |
| 中级符咒 | ✅ |
| 高级符咒 | ✅ |
| 专属符咒 | ✅ |
| 御魂 | ❌ |
| 恶评 | ❌ |
| 式神 | ❌ |

## 🛠️ 实现备注

### 涉及文件

- `shared/game/effects/YokaiEffects.ts` - 木魅效果实现
- `shared/types/cards.ts` - 卡牌类型判定

### 关键函数

```typescript
registerEffect('木魅', async (ctx) => {
  const { player } = ctx;
  
  const revealed: CardInstance[] = [];
  const spellsFound: CardInstance[] = [];
  
  // 从牌库顶开始展示，直到找到3张阴阳术或牌库空
  while (spellsFound.length < 3 && player.deck.length > 0) {
    const card = player.deck.shift()!;
    revealed.push(card);
    
    // 检查是否为阴阳术
    if (isSpellCard(card)) {
      spellsFound.push(card);
    }
    
    ctx.addLog?.(`📖 展示: ${card.name}`);
  }
  
  // 阴阳术入手
  for (const spell of spellsFound) {
    player.hand.push(spell);
    revealed.splice(revealed.indexOf(spell), 1);
  }
  
  // 其余弃置
  for (const card of revealed) {
    player.discard.push(card);
  }
  
  ctx.addLog?.(`🎴 木魅 → 获得${spellsFound.length}张阴阳术，弃置${revealed.length}张`);
  
  return { success: true, message: `获得${spellsFound.length}张阴阳术` };
});

function isSpellCard(card: CardInstance): boolean {
  return card.cardType === 'spell';
}
```

### 注意事项

- **多人专属**：此卡仅在5-6人游戏使用（`multiPlayer = count = 2`）
- **展示公开**：展示过程中牌面信息对所有玩家可见
- **弃置可能触发效果**：需要检查被弃置的牌是否有【被弃置时】触发
- **牌库循环**：不触发弃牌堆洗回牌库（仅从当前牌库搜索）
