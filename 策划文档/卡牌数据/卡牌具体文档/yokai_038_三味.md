## 🔍 效果分解

### 步骤拆解

**御魂效果（打出时）**：
1. **计算已使用数量**：统计本回合**在三味打出之前**已使用的「鬼火」牌和阴阳术的总数量 X
2. **伤害增益**：伤害 + (2 × X)

**触发效果【触】（被弃置时）**：
3. **抓牌+3**：从牌库顶抓取3张牌

### 关键规则

| 规则 | 说明 |
|------|------|
| 统计范围 | **本回合已使用**的「鬼火」牌和阴阳术 |
| 统计时机 | 三味**打出时**即时计算，不包含三味自身 |
| 「鬼火」牌 | 指 `subtype` 包含 `鬼火` 的妖怪卡（如灯笼鬼、日女巳时等） |
| 阴阳术 | 指 `type === 'spell'` 的卡牌（基础术式、中级符咒、高级符咒） |
| 伤害计算 | 每张符合条件的牌 +2 伤害 |
| 【触】弃置 | **主动弃置**时触发，回合结束清空手牌**不触发** |

### 「鬼火」牌定义

## 🛠️ 实现备注

- `shared/game/effects/TriggerEffects.ts` - 注册弃置触发效果
- `shared/game/effects/YokaiEffects.test.ts` - TDD测试用例

### 实现代码

```typescript
// 御魂效果
registerEffect('三味', async (ctx) => {
  const { player, gameState } = ctx;
  
  // 统计本回合已使用的阴阳术和「鬼火」牌数量
  const usedCards = gameState.turnHistory?.playedCards || [];
  const count = usedCards.filter(card => 
    card.cardType === 'spell' || 
    (card.subtype && card.subtype.includes('鬼火'))
  ).length;
  
  // 计算伤害加成
  const damageBonus = count * 2;
  player.damage += damageBonus;
  
  return { 
    success: true, 
    message: count > 0 
      ? `伤害+${damageBonus} (使用了${count}张阴阳术/鬼火牌)` 
      : '无伤害加成 (未使用阴阳术/鬼火牌)'
  };
});

// 弃置触发效果
registerDiscardTrigger('三味', async (ctx) => {
  const { player, discardType } = ctx;
  
  // 只有主动弃置才触发
  if (discardType !== 'active') return { success: false };
  
  // 抓牌+3
  drawCards(player, 3);
  
  return { success: true, message: '三味被弃置，抓牌+3' };
});
```

### 注意事项

1. **统计时机**：三味打出时统计，不包含三味自身
2. **「鬼火」牌判定**：检查 `subtype` 是否包含「鬼火」字符串
3. **弃置类型**：需要区分主动弃置和回合结束清空
4. **日志格式**：
   - 御魂: `🎴 ${player.name} 打出三味，伤害+${bonus}`
   - 弃置: `🃏 三味被弃置，${player.name} 抓牌+3`

需要实现以下系统：
1. **回合历史记录**：`gameState.turnHistory.playedCards` 记录本回合已打出的牌

interface TurnHistory {
