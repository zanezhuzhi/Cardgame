# 鬼王卡牌数据

> **版本**: v0.3.0（基于 202005_阴阳师+桌游+游戏反馈.xlsx）  
> **总计**: 10张，分3个阶段（Ⅰ/Ⅱ/Ⅲ）  
> **说明**:  
> - **来袭效果** = 翻出此鬼王时立即执行，影响所有玩家  
> - **御魂效果** = 打出此鬼王的御魂牌时触发  
> - **🔷** = 5人以上专属（多人游戏标记=1）

| **关键词** | 御魂 |

## 阶段 Ⅲ

### 地震鲶
| 属性 | 值 |
|------|-----|
| **阶段** | Ⅲ |
| **生命** | 13 |
| **声誉** | +5 |
| **关键词** | 御魂/妨害 |
| **🔷多人** | 否 |

**御魂效果**: 所有对手玩家将手牌数弃至3张，你可以获得其中生命值最高的御魂卡并将其置入手牌。（同为最高则你可以选择）

**来袭效果**: 每位玩家在清理阶段补充完手牌后，随机将一张手牌背面朝上放置于各自的阴阳师下。当此鬼王被击败后将阴阳师下的所有卡牌弃置。

## 特殊机制说明

### 【自】回合开始回收
以下鬼王具有回合开始时从弃牌堆回收到手牌的能力：
- **蜃气楼**（阶段Ⅱ）
- **荒骷髅**（阶段Ⅱ）
- **贪嗔痴**（阶段Ⅲ）

### 【触】回合结束归底
- **麒麟**（阶段Ⅰ）：回合结束时，若此牌在弃牌区，将其置入牌库底
