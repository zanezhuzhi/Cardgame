## 一、游戏概述

### 1.1 游戏类型
- DBG（Deck Building Game）卡牌构筑游戏
- 3-6人对战

### 1.2 游戏模式

| 模式 | 人数 | 说明 |
|------|:----:|------|
| **标准模式** | 3-4人 | 移除带「纸人🔷」标记的卡牌 |
| **多人模式** | 5-6人 | 使用全部卡牌 |

> 达到对应人数即可开始游戏

### 1.3 胜利条件

**游戏结束触发**：
1. 当游荡妖怪牌库中翻出新的妖怪不足以补充至6张时，游戏结束
2. 当所有鬼王均被击败时，游戏立即结束

**胜利判定**：
1. 首先比较【声誉】总数，高者胜
2. 声誉相同则比较**牌库张数**
3. 再相同则判为**平局**

## 二、核心概念

### 2.1 卡牌区域

每位玩家有以下区域：

| 区域 | 说明 | 可见性 |
|------|------|--------|
| **手牌** | 当前持有的牌 | 仅自己可见 |
| **待抓取牌（牌库）** | 面朝下的牌堆 | 不可见 |
| **弃牌堆** | 打出/弃置的牌 | 公开 |
| **式神区** | 已召唤的式神 | 公开 |
| **已打出区（played）** | 本回合打出的牌，清理阶段移入弃牌堆 | 公开 |

> **重要概念：牌库**  
> 牌库 = 待抓取牌 + 手牌 + 弃牌堆  
> 打出的牌、退治的目标均进入弃牌堆，成为牌库的一部分。  
> **超度的卡牌移出游戏，不属于牌库，不计入声誉结算。**

> **循环机制**：待抓取牌耗尽时，将弃牌堆洗牌成为新的待抓取牌（手牌不变）。

### 2.2 公共区域

| 区域 | 说明 |
|------|------|
| **游荡妖怪展示区** | 场上展示的6张妖怪，可被退治 |
| **游荡妖怪牌库** | 面朝下的妖怪牌堆 |
| **鬼王区** | 当前阶段的鬼王 |
| **恶评牌库** | 农夫+武士混洗的牌堆 |
| **阴阳术供应堆** | 基础术式/中级符咒/高级符咒 |
| **式神供应堆** | 可获取的式神 |
| **超度区** | 移出游戏的卡牌（公共） |

## 三、资源系统

### 3.1 鬼火
| 属性 | 说明 |
|------|------|
| **获取** | 每回合开始时固定获得1点 |
| **使用** | 消耗鬼火以发动式神技能 |
| **上限** | **5点**（超出部分自动消失） |
| **回合结束** | **保留**（不清零） |

### 3.2 伤害
| 属性 | 说明 |
|------|------|
| **获取** | 由阴阳术、妖怪效果、式神技能产生 |
| **使用** | 在行动阶段**一次性**分配给目标 |
| **回合结束** | **清零** |

### 3.3 声誉
| 属性 | 说明 |
|------|------|
| **结算时机** | 游戏结束时统计 |
| **计算范围** | 牌库（含弃牌堆）+ 式神区 |

**声誉来源**：

| 卡牌类型 | 声誉范围 | 说明 |
|----------|:--------:|------|
| 招福达摩 | +1 | 初始卡 |
| 妖怪卡 | 0 ~ +2 | 退治后进入弃牌堆 |
| 鬼王卡 | +3 ~ +5 | 击败后进入弃牌堆 |
| 式神卡 | +1 ~ +3 | 在式神区 |
| 高级符咒 | +1 | 在牌库中 |
| 恶评卡 | -1 / -2 | 农夫-1，武士-2 |
| 超度的卡 | **不计入** | 已移出游戏 |

## 五、游戏流程

### 5.1 游戏准备

#### 步骤1：准备公共牌库
1. 根据人数确定游戏模式（3-4人标准/5-6人多人）
2. 移除对应的🔷牌（标准模式移除32张妖怪+3张鬼王+6张式神）
3. 洗混游荡妖怪牌库、恶评牌库
4. 按阶段排列鬼王牌库（Ⅰ→Ⅱ→Ⅲ，麒麟在最上方）

#### 步骤2：每位玩家准备
1. 选择代表自己的**阴阳师**（仅形象，无技能差异）
2. 获得**1颗鬼火骰子**
3. 获得初始牌库：**3张招福达摩 + 6张基础术式 = 9张**
4. 洗混后抓**5张**作为起始手牌

#### 步骤3：选取式神
1. 从式神供应堆顶抓4张
2. 选择1张作为初始式神（式神上限3个）
3. 其余3张放回式神供应堆底

#### 步骤4：场地准备
1. 翻开**5张**游荡妖怪到展示区（加上牌库顶牌共**6张**可见）
2. 翻开第一张鬼王（麒麟，无来袭效果）
3. 决定起始玩家

---

### 5.2 回合流程

每个回合按以下阶段依次执行：

```
┌─────────────────────────────────────────────────┐
│  1. 鬼火阶段    → 鬼火+1（上限5）                 │
├─────────────────────────────────────────────────┤
│  2. 式神调整阶段 → 可获取/更换式神（可选）         │
├─────────────────────────────────────────────────┤
│  3. 行动阶段    → 打牌、使用技能、分配伤害        │
├─────────────────────────────────────────────────┤
│  4. 清理阶段    → 弃牌、补牌、补充妖怪、鬼王检查  │
└─────────────────────────────────────────────────┘
```

#### 阶段1：鬼火阶段
- 当前回合玩家获得**鬼火+1**（上限5）
- 清空上回合残余的**临时增益（TempBuff）**

#### 阶段2：式神调整阶段（可选）
- 可以从式神供应堆获取新式神（上限3个）
- 可以更换现有式神
- 通过打出「地藏像」妖怪卡获取式神

#### 阶段3：行动阶段
可以任意顺序执行以下动作：
1. **打出手牌**：阴阳术或妖怪卡
2. **使用式神技能**：消耗鬼火发动【启】技能
3. **获得基础术式**：每回合可免费获得1张
4. **分配伤害**：**只能进行一次**，详见"伤害分配规则"

#### 阶段4：清理阶段
1. 将本回合已打出的牌 + 剩余手牌 → 弃牌堆
2. 【御魂/持续】类型卡可选择抓回手上
3. 从牌库抽取**5张**作为下回合手牌
4. 从游荡妖怪牌库补充妖怪至**6张**
5. 如果本回合击杀了鬼王 → 翻出下一张鬼王 → 执行【鬼王来袭】
6. 回合结束，清空伤害

## 六、核心规则详解

### 6.1 伤害分配规则（重要！）

1. **只可以进行一次分配伤害**
2. 分配后产生的伤害**无法造成后续伤害**
3. 每1点伤害扣除指定目标1点生命
4. **上一个目标生命降至0之前，无法指定新目标**
5. 生命为0的单位可选择：**退治**（置入弃牌堆）或**超度**（移出游戏）
6. **回合进行中，游荡妖怪不会因离场而刷新**
7. 行动阶段结束时，所有单位生命恢复至上限，未使用伤害清空

### 6.2 退治与超度

| 操作 | 说明 | 卡牌去向 | 声誉 |
|------|------|----------|------|
| **退治** | 将生命降为0的目标获取 | 进入自己的弃牌堆 | **计入** |
| **超度** | 将卡牌移出游戏 | 进入公共超度区 | **不计入** |

### 6.3 鬼王系统

**鬼王阶段**：Ⅰ → Ⅱ → Ⅲ

**鬼王来袭**：当新鬼王翻出时，立即执行来袭效果，影响所有玩家

**御魂效果**：鬼王卡本身也是御魂牌，打出时触发御魂效果

**特殊机制**：
- 部分鬼王具有【自】效果：回合开始时从弃牌堆回收到手牌
- 麒麟【触】：回合结束时从弃牌堆回到牌库底

### 6.4 式神技能类型

| 标记 | 名称 | 说明 |
|------|------|------|
| **【启】** | 主动技 | 消耗鬼火发动 |
| **【触】** | 触发技 | 满足条件时自动触发 |
| **【自】** | 被动触发 | 满足条件时自动执行 |
| **【永】** | 永久被动 | 只要在场就生效 |
| **【妖】** | 妖怪专属 | 作为游荡妖怪时的效果 |

### 6.5 临时增益系统（TempBuff）

部分卡牌效果会产生**本回合有效**的临时增益：

| 效果类型 | 示例 |
|----------|------|
| 阴阳术伤害加成 | 山童【怪力】：前2张阴阳术+1伤害 |
| 式神技能伤害加成 | 针女：每次使用式神技能时伤害+2 |
| 御魂效果翻倍 | 轮入道：执行另一张御魂效果两次 |
| 鬼火消耗减少 | 涅槃之火：式神技能鬼火消耗-1 |

> 所有TempBuff在回合开始时清空

## 九、开发实现要点

> 以下是开发过程中确认的实现细节，供后续开发者参考

### 9.1 牌库结构

```typescript
interface PlayerState {
  deck: Card[];        // 待抓取牌
  hand: Card[];        // 手牌
  discard: Card[];     // 弃牌堆
  played: Card[];      // 本回合已打出（清理时移入discard）
  shikigami: Shikigami[];  // 式神区
  ghostFire: number;   // 鬼火（0-5）
  damage: number;      // 本回合伤害（回合结束清零）
  tempBuffs: TempBuff[];   // 临时增益（回合开始清空）
}
```

### 9.2 招福达摩特殊处理

- 招福达摩是**妖怪卡**，但**不进入游荡妖怪牌库**
- 仅用于游戏初始发给每位玩家（每人3张）
- 96张妖怪卡 = 18张招福达摩（初始） + 78张游荡妖怪

### 9.3 🔷纸人标记含义

🔷数字表示"该卡种中有几张是多人专属"，而非"整个卡种都是多人专属"

例：
- 唐纸伞妖：总数2张，🔷=1 → 1张普通+1张多人专属
- 镜姬：总数2张，🔷=2 → 2张都是多人专属（3-4人完全不用）
- 赤舌：总数2张，🔷=0 → 2张都是普通（3-4人全用）

### 9.4 恶评牌库耗尽处理

当恶评牌库剩余为0时，"获得1张恶评"效果默认获得**农夫**（无限供应）

### 9.5 效果引擎标签

```typescript
// 效果类型枚举
type EffectType = 
  | 'DRAW'           // 抓牌
  | 'DAMAGE'         // 伤害
  | 'GHOST_FIRE'     // 鬼火
  | 'DISCARD'        // 弃置
  | 'EXILE'          // 超度
  | 'CHOICE'         // 选择一项
  | 'CONDITIONAL'    // 条件效果
  | 'HINDRANCE'      // 妨害
  | 'KILL_YOKAI'     // 退治妖怪
  | 'VIEW_DECK_TOP'  // 查看牌库顶
  // ...
```
