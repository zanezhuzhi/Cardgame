## 一、游戏概述

### 1.1 游戏类型
- DBG（Deck Building Game）卡牌构筑游戏
- 2-6人对战

### 1.2 胜利条件
- 游戏一共有2种结束方式：
  1. 当游荡妖怪牌组中翻出新的妖怪不足以补充至6张时，游戏结束
  2. 当所有鬼王均被击败时，游戏立即结束
- 声誉最高的玩家成为获胜者

## 二、资源系统

### 2.1 鬼火
- **获取方式**: 每回合固定获得1点
- **使用方式**: 扣除鬼火以发动式神技能
- **回合结束时**: 保留，**上限为5**

### 2.2 伤害
- **获取方式**: 由式神技能或阴阳术增加
- **使用方式**: 在行动阶段末期分配给目标
- **回合结束时**: 清零

### 2.3 声誉
- 在游戏结束时，最高声誉的玩家赢得游戏
- 通过卡牌累积

**结算方式**：游戏结束时，统计玩家**牌库**与**式神区**中所有卡牌的 charm 值总和。

> **牌库** = 待抓取牌（deck）+ 手牌（hand）+ 弃牌堆（discard）  
> 打出的牌、退治的目标均进入弃牌堆，成为牌库的一部分。  
> 超度的卡牌移出游戏，**不属于牌库，不计入**声誉结算。
>
> **循环机制**：待抓取牌耗尽时，将弃牌堆重新洗牌成为待抓取牌（此时手牌不变）。

**声誉来源**：

| 卡牌 | 放置区域 | charm |
|------|---------|-------|
| 退治妖怪（御魂） | 牌库（弃牌堆） | 0~2 |
| 击败鬼王 | 牌库（弃牌堆） | 3~5 |
| 式神（在场） | 式神区 | 1~3 |
| 高级符咒 | 牌库 | +1 |
| 恶评卡 | 牌库（弃牌堆） | -1 / -2 |
| 超度的卡 | 移出游戏（**不计入**） | — |

## 四、游戏阶段

### 4.1 鬼火阶段
- 当前回合玩家获得 **鬼火+1**
- 鬼火**最高累积5点**

### 4.2 式神调整阶段
- 玩家可以决定是否调整式神
- 式神上限为3张

### 4.3 行动阶段
- 打出手牌中的阴阳术和妖怪
- 退治妖怪或鬼王（伤害 ≥ 生命值 → 退治成功）
- **只可以进行一次分配伤害**

### 4.4 清理阶段
1. 弃牌
2. 补牌至5张
3. 清空伤害
4. 补充妖怪至6张
5. 鬼王检查

## 五、关键字详解

### 5.1 [鬼火]
- **鬼火+X**：增加X点鬼火，**上限为5**
- **鬼火-X**：消耗X点鬼火

### 5.2 [伤害]
- **伤害+X**：本回合可分配伤害增加X点

### 5.3 [抓牌]
- **抓牌+X**：从牌库顶抓取X张卡牌

### 5.4 [弃置]
- 将卡牌从手牌移到弃牌区

### 5.5 [超度]
- 将卡牌彻底移出游戏

### 5.6 [妨害]
- 影响其他玩家的效果

## 六、初始设置

### 步骤1：选择阴阳师
- 每名玩家选择代表自己的阴阳师
- 拿取1颗鬼火骰子

### 步骤2：选取式神
- 抓取式神牌堆顶4张，选择2张保留

### 步骤3：拿取基础牌组
- 6张基础阴阳术 + 4张招福达摩
- 洗切后抓5张作为起始手牌

## 八、开发摘要（从规则书提取）

> 以下信息摘自《御魂传说》规则书.docx，对开发有重要参考价值

### 8.1 设计思路
- 降低基础规则复杂度，提升与游戏的结合点
- 式神强度主要体现在交互上，降低绝对强度对卡组构建的节奏影响
- 统一卡牌种类为【御魂】【阴阳术】【恶评】三类
- 伤害结算固定为一次性流程，避免频繁回退和复盘

### 8.2 价值公式（卡牌平衡参考）
```
【价值】= 【生命值】× 生命价值参数（分段）
       = 【咒力】× 伤害参数 
       + 【抓牌】× 手牌参数 
       + 【符咒数】× 胜利点参数 
       + 【鬼火】× 鬼火参数 
       + 特殊效应价值
```

### 8.3 御魂关键词（官方定义）

| 关键词 | 效果 |
|--------|------|
| **退治/获得** | 将战场目标置入自己弃牌堆，无法立刻使用 |
| **生命N** | 在场上时，需造成N点伤害方可退治 |
| **符咒N** | 游戏结束时累加的胜利点数 |
| **抓牌+1** | 立刻抓牌库顶牌 |
| **咒力+1** | 对场上目标造成1点伤害 |
| **鬼火+1** | 获得1点鬼火（超出5点自动消失） |
| **弃置** | 将卡牌面朝上置入弃牌堆最上方 |
| **超度** | 将卡牌置入公共超度区（移出游戏） |
| **展示** | 翻至正面供所有人检视，然后归还原状态 |
| **御魂/妨害** | 对其他玩家造成影响，可能触发反制 |
| **御魂/持续** | 清理阶段可选择抓回手上而非弃置 |

### 8.4 分配伤害规则（重要！）
1. **只可以进行一次分配伤害** - 确保所有咒力统计在内
2. 分配后产生的咒力**无法造成后续伤害**
3. 每1点咒力扣除指定目标1点生命
4. **上一个目标生命降至0之前，无法指定新目标**
5. 可选择在**结算前**或**结算后**一次性分配完毕
6. 生命为0的单位可选择：置入弃牌堆 或 超度
7. **回合进行中，游荡妖怪不会因离场而刷新**
8. 行动阶段结束时，所有单位生命恢复至上限，未使用咒力清空

### 8.5 清理阶段详细流程
1. 将所有已打出旁置的牌+剩余手牌 → 弃牌堆
2. **【御魂/持续】类型卡可选择抓回手上**
3. 从牌库顶抽取5张卡作为下回合手牌
4. 从游荡妖怪牌组翻出新妖怪补充至6张
5. 如果本回合击杀了鬼王 → 翻出下一张鬼王 → 执行【鬼王来袭】

### 8.6 胜利条件
- **游戏结束触发**：
  1. 场上所有鬼王被退治
  2. 游荡妖怪补充不足6张
- **胜利判定**：
  1. 首先比较【符咒】总数，高者胜
  2. 符咒相同则比较**牌张数量**
  3. 再相同则判为**平局**
