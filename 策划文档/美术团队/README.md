## 🔄 工作流程

1. **需求分析** → 查看美术需求清单.md
2. **风格确认** → 参考风格指导.md、SSR式神设计规范.md
3. **资源制作** → 输出到 `角色卡牌设计/` 对应目录
4. **质量审核** → 与制作人/开发团队确认
5. **交付对接** → 将确认的资源交付开发团队
