## 🎨 高级用法

### 1. 根据游戏状态调整氛围

```typescript
// 战斗开始时增强氛围
function startBattle() {
  atmosphereLayerRef.value.updateConfig({

    ghostFire: false,  // 鬼火消失

    fog: false
```

```typescript
// 音效管理器
const sfxManager = {
  hover: new Audio('/sfx/hover.wav'),
  click: new Audio('/sfx/click.wav'),

  damage: new Audio('/sfx/damage.wav'),

  play(effect: string, volume = 0.5) {
    const audio = this[effect]
    if (audio) {
      audio.currentTime = 0
      audio.volume = volume
      audio.play().catch(err => console.warn('音效播放失败:', err))
    }
  }
}
```

```typescript
// 添加性能监控
function monitorPerformance() {
  const fpsCounter = document.createElement('div')
  fpsCounter.className = 'fps-counter'
  document.body.appendChild(fpsCounter)
  
  let lastTime = performance.now()
  let frameCount = 0
  
  function updateFPS() {
    const now = performance.now()
    frameCount++
    
    if (now - lastTime >= 1000) {
      fpsCounter.textContent = `FPS: ${frameCount}`
      frameCount = 0
      lastTime = now
    }
    
    requestAnimationFrame(updateFPS)
  }
  
  updateFPS()
}
```

## 📊 性能优化建议

### 1. 粒子系统优化
- 限制粒子数量（樱花 < 50，鬼火 < 10）
- 使用 `ParticleContainer` 提升性能
- 粒子离开屏幕后立即回收

### 2. 动画优化
- 使用 CSS transform 和 opacity（GPU加速）
- 避免频繁的布局重排
- 使用 `requestAnimationFrame` 同步动画

### 3. 内存优化
- 及时清理销毁的组件
- 使用对象池复用粒子
- 避免内存泄漏

### 4. 低配模式
```typescript
function enableLowPerformanceMode() {
  atmosphereLayerRef.value.updateConfig({
    sakura: false,
    ghostFire: false,
    mountains: true,
    fog: false
  })
}
```

## 🐛 常见问题

### Q1: 氛围层不显示
- 检查 Pixi.js 是否正确安装
- 确认容器尺寸正确
- 查看控制台是否有错误

### Q2: 性能问题
- 降低粒子数量
- 关闭不必要的特效
- 启用低配模式

### Q3: 动画不流畅
- 检查是否有过多的 DOM 操作
- 优化 CSS 选择器
- 使用 GPU 加速属性
