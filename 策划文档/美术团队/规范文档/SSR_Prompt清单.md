## 卡片1：妖刀姬 (Yato) - 杀戮之刃

### 设计Prompt

```
【角色设计】
年轻女性战士，16-18岁，身姿矫健
黑色长发及腰，发尾随刀气飘动
表情冷峻，眼神锐利如刀

【服装设计】
深色和风战斗服，黑色(#151525)为主色调
银白色金属护肩和腰带
衣袖有破损战斗痕迹
腰间佩戴黑色刀鞘，刀鞘有暗红纹路

【核心道具】
巨型武士刀，刀身血红渐变
刀身刻有诅咒符文，发出暗红光芒
刀长为角色身高1.2倍

【姿态与动作】
拔刀斩姿态，身体向右倾斜
左脚在前，右脚在后发力
刀光从右上方划向左下方
周围有血红色刀气轨迹

【场景氛围】
深紫背景，樱花瓣被刀气卷入
刀刃发出血红光芒
金色粒子点缀，增强超凡感

【禁止要素】
禁止粉红色服装
禁止短发造型
禁止粉色系妖刀
```

### 详细描述（供美术设计师参考）

```
构图：人物占画面70%，居中偏右，刀光向左延伸
头身比：7头身
色彩重点：血红刀光 vs 黑色服装 vs 金色边框
```

## 卡片2：大天狗 (Tengu) - 羽刃暴风

### 设计Prompt

```
【角色设计】
男性形态，体型健硕但不臃肿
肤色深红，脸部特征夸张化（长鼻）
表情威严但不狰狞

【鸟类特征】
巨大乌鸦翅膀，完全展开
翅膀羽毛以黑色为主，边缘有紫色光泽
翅膀占据画面左右两侧

【服装设计】
传统天狗装束，深色狩衣
宽大袖子随风飘动
金色束腰带
脚部腾空，悬浮姿态

【核心元素】
周围环绕金色风刃效果
有落叶/碎石被卷起
云雾缭绕腿部

【姿态与动作】
展翅高飞，双臂张开
身体微微后仰，增强气势
周围风暴云雾环绕

【场景氛围】
风暴云雾背景，深紫色
金色风刃点缀
翅膀羽毛细节丰富

【禁止要素】
禁止黑色羽衣造型
禁止佩戴面具
禁止完全拟人化缺少妖怪感
```

### 详细描述（供美术设计师参考）

```
构图：人物居中，翅膀展开占满画面宽度
头身比：7.5头身（强调威严感）
色彩重点：黑色羽翼 vs 深红肤色 vs 金色风刃
```

## 卡片3：酒吞童子 (Shuten) - 酒之狂鬼

### 设计Prompt

```
【角色设计】
鬼族男性，体型魁梧有力
面容粗犷但有魅力，胡须点缀
皮肤古铜色，带有鬼族纹路

【鬼角设计】
粗壮双角，向后上方弯曲
角表面有年轮状纹路
角根部连接处有金饰

【服装设计】
鬼族战甲风格，裸露部分肌肉
深红色皮甲 + 暗金金属
肩部和前臂有骨刺装饰
腰间悬挂小酒壶

【核心道具】
巨大酒葫芦，悬浮于身侧
葫芦表面有裂纹，发出金色酒光
葫芦比人头略大

【姿态与动作】
微微醉酒摇晃姿态，但气势不减
一手叉腰，一手指向酒葫芦
表情狂放不羁

【场景氛围】
深红/暗紫火焰背景
酒气蒸腾如云
金色光芒从葫芦散发

【禁止要素】
禁止红色短发造型
禁止手持酒瓶
禁止完全年轻化面容
```

### 详细描述（供美术设计师参考）

```
构图：人物占画面65%，酒葫芦作为重要视觉元素
头身比：6.5头身（魁梧感）
色彩重点：金色酒光 vs 深红战甲 vs 暗紫火焰
```
