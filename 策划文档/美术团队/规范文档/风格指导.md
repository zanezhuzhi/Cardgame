## 一、核心设计理念

### 1.1 指导原则

《御魂传说》是《阴阳师》IP的衍生卡牌产品，美术风格必须与原作保持高度一致性，传达"神秘、精致、东方"的视觉体验。

**三大核心原则**：
1. **文化根基**：以东方妖怪文化为核心，融入日式和风美学
2. **品质感**：通过精细细节和高级配色传递品质感
3. **沉浸感**：通过动态元素和氛围营造强化代入体验

### 1.2 阴阳师UI设计的核心逻辑

> 通过暗色调背景衬托华丽金色元素，既符合"阴阳师"主题的神秘感，又保证了功能的清晰性。

**核心公式**：
```
深色背景 + 金色边框/文字 + 动态点缀 = 阴阳师视觉体系
```

## 三、金色边框规范

### 3.1 使用场景

| 场景 | 效果要求 | 参考案例 |
|------|----------|----------|
| SSR/SP稀有度标识 | 浮雕质感 + 光晕效果 | 阴阳师SSR框 |
| 重要弹窗边框 | 金色描边 + 浮雕阴影 | 式神录弹窗 |
| 按钮边框 | 金色描边 + 悬停发光 | 召唤按钮 |
| 卡牌边框 | 金属质感 + 装饰花纹 | 卡牌立绘框 |

### 3.2 效果规范

- **浮雕效果**：金色边框采用内阴影/外发光组合，呈现立体金属质感
- **光效反馈**：交互时金色边框应有光效动画（如流光、闪烁）
- **渐变处理**：金色采用 `#d4a574` → `#ffd700` 渐变，避免纯色平涂

## 四、妖怪设计规范

### 4.1 核心原则

> 以东方妖怪文化为核心，通过非人元素增强妖怪属性，避免角色过于"人类化"。

### 4.2 妖怪感营造要素

| 要素 | 实现方式 | 参考案例 |
|------|----------|----------|
| 动物特征 | 兽耳、尾巴、爪牙 | 山兔、孟婆 |
| 器物化身体 | 灯笼、伞、面具成精 | 灯笼鬼、提灯小僧 |
| 异化造型 | 扭曲形态、异色皮肤 | 络新妇、般若 |
| 文化符号 | 和风服饰、妖怪纹样 | 各式神原皮 |

### 4.3 设计禁区

- ❌ 角色过于"人类化"（缺少妖怪特征）
- ❌ 西式审美元素（迪士尼风格、欧美奇幻）
- ❌ 过于可爱/萌系（丢失妖怪的神秘感）
- ❌ 纯恐怖风格（不符合卡牌游戏的轻松氛围）

## 五、UI布局规范

### 5.1 分层式布局原则

```
┌─────────────────────────────────────────┐
│  资源栏层（勾玉、体力、声誉等）           │  ← 顶部固定
├─────────────────────────────────────────┤
│                                         │
│  主内容层（地图、战斗、卡牌展示区）        │  ← 中央核心
│                                         │
├─────────────────────────────────────────┤
│  功能栏层（菜单、社交、任务提示）          │  ← 底部固定
└─────────────────────────────────────────┘
```

### 5.2 对称与平衡

- **中心对称**：重要页面（召唤、式神录）采用中心对称布局
- **功能分区**：左（任务）/中（内容）/右（活动）/底（主菜单）
- **视觉重心**：核心内容置于画面中心或黄金分割点

### 5.3 可读性原则

- 深底配亮金色或白色文字
- 功能色系统保证信息可读性
- 避免在同一区域使用超过3种高饱和度颜色

## 六、动态元素规范

### 6.1 允许的动态元素

| 元素 | 效果描述 | 使用场景 |
|------|----------|----------|
| 樱花飘落 | 粉白花瓣缓慢飘落，带旋转 | 主界面、召唤界面 |
| 云雾流动 | 半透明烟雾缭绕 | 背景层次 |
| 鬼火浮动 | 蓝色光点缓慢移动 | 技能释放、特效 |
| 金色流光 | 金色边框光效流动 | 稀有卡、弹窗边框 |
| 波纹扩散 | 同心圆波纹扩散 | 技能命中、buff生效 |

### 6.2 动态效果规范

- **频率**：动态元素应为"点缀"，不应喧宾夺主
- **性能**：避免过于复杂的粒子效果，保证流畅度
- **风格**：符合和风/妖怪主题，避免科幻/赛博朋克元素

## 七、文化元素参考

### 7.1 可融入的传统元素

| 类别 | 元素 | 应用场景 |
|------|------|----------|
| 建筑 | 神社、鸟居、灯笼 | 背景装饰 |
| 纹样 | 御魂纹、阴阳鱼、樱花纹 | 卡牌边框、图标底纹 |
| 服饰 | 狩衣、巫女服、铠甲 | 角色设计 |
| 道具 | 符咒、铃铛、面具 | UI图标、特效 |
| 自然 | 枫叶、竹林、月光 | 氛围装饰 |

### 7.2 传统元素"再设计"原则

1. **保留核心结构**：如交叉前襟、方形轮廓
2. **现代材质处理**：通过材质、色彩赋予现代感
3. **文化细节点缀**：通过纹身、道具深化角色叙事

## 八、与《阴阳师》的差异处理

### 8.1 允许的差异

| 方面 | 差异处理 | 原因 |
|------|----------|------|
| 主色调 | 深紫蓝替代深黑/暗红 | 卡牌游戏需要更强的辨识度 |
| UI材质 | 半透明磨砂玻璃 | 现代化UI趋势，提升科技感 |
| 卡牌比例 | MTG 5:7标准 | 卡牌游戏功能性需求 |

### 8.2 禁止的差异

- ❌ 替换金色主色调为其他颜色
- ❌ 采用欧美奇幻风格的角色设计
- ❌ 使用非和风的装饰元素（如圣诞、雪地）
- ❌ 简化或取消妖怪特征

## 九、版权合规规范（新增 2026-03-22）

### 9.1 版权风险识别

| 风险类型 | 示例 | 规避方式 |
|----------|------|----------|
| 直接复制 | 使用原版游戏立绘 | ❌ 严格禁止 |
| 高度相似 | 近似构图/配色/姿态 | ❌ 规避相似度 |
| 文化公有领域 | 民间传说名称/形象 | ✅ 可使用 |
| 文化符号 | 妖怪类型、传统服饰 | ✅ 可参考 |

### 9.2 合规设计原则

> **核心原则**：如果一个不了解原游戏的人看到新图，不会联想到《阴阳师》，则为合规。

**可保留元素**：
- 式神名称（民间传说/文化元素）
- 基础妖怪类型特征（非人元素、器物化身体）
- 文化符号（如鬼面、和风服饰轮廓）

**必须原创**：
- 角色姿态和构图
- 服装设计和配色
- 武器/道具造型
- 面部特征和表情
- 光效和特效设计

### 9.3 审核检查清单

- [ ] 无官方立绘构图相似
- [ ] 无官方服装设计雷同
- [ ] 无官方配色方案照搬
- [ ] 无标志性动作复制
- [ ] 角色整体辨识度符合预期
